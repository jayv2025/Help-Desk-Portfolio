# Installing Windows 11 VM

---
	1. Download ISO windows 11 Enterprise at Microsoft website
		a. English 
		b. ISO-Enterprise Download
		c. 64-bit edition

---

	2. Virtualbox
		a. New 
		b. Name: windows 11 Enterprise 
		c. ISO image (just downloaded)
		d. Skip unintended installation
		e. Hardware: if have resources up to 8GBs and 4 cores
			i. Base memory
			ii. Processor
		f. Virtual hard disk
		g. Finish 

---

	3. Start 

---

	4. Accept the license terms 

---
	5. Custom install windows only 
		a. Default drive
		b. Next 

---

	6. Continue configurations 

--

	7. Domain join instead 
		a. Enter name (username) 
		b. Password 
		c. Security questions 
		d. Uncheck privacy settings 

---

	8. Log in 

---

	9. Devices 
		a. Insert guest additions CD image

--- 

	10. File explorer
		a. Double click on CD drive 
		b. Right click on VBoxWindowsAdditions-amd64
			i. Run as admin
			ii. Next 
			iii. Defaults 
			iv. Install
			v. Reboot now 
		c. View 
			i. Auto-resize guest display
			ii. Full-screen mode

# Installing VirtualBox 

This section demonstrates how to install the **Oracle VirtualBox** hypervisor on a host system. 
***(Note: Virtualbox was already installed prior to lab )***

---


### Step 1: Download VirtualBox
1. Navigate to: https://www.virtualbox.org/wiki/Downloads
2. Select the appropriate package for your operating system 
    - Example: **Windows hosts** for a Windows machine
3. Click the download link 
4. Choose a save location (e.g., **Desktop**)
5. Click **Save**

--- 

### Step 2 Install VirtualBox
Once the download is complete: 
1. Double-click the installer file 
2. When prompted by **User Account Control (UAC)**, click **Yes**
3. In the setup wizard:
    - Leave the **default installation location** -> Click **Next**
    - Accept installation prompts -> Click **Yes**
    - Continue through the setup steps
4. Click **Install**

---
 
### Step 3 Complete installation

1. Wait for the installation to finish 
2. Click **Finish**

VirtualBox should now be successfully installed and ready to use.
